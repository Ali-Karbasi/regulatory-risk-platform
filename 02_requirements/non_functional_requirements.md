# Non_Functional Requiremnts

## NFR-1: Data Integrity
all records must include system-generated timestamps and user identifiers to support audit traceability.

## NFR-2: Performance
executive dashboard msut load within 5 seconds for datasets up to 100,000 records.

## NFR-3: Security
access to risk and regulatory data must be role-based, ensuring only authorized users can modify records.

## NFR-4: Maintainability
tha data model must support future regulatory categories and business units without requiring structural redesign.
